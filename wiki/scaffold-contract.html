Scaffold Contract

Summary

  • Status: active product convention
  • Source index: wiki/sources.html
  • Wiki agent guide: wiki/AGENTS.html
  • Default styling: app-served /assets/wiki.css
  • Runtime boundary: Localhost Tooling

Decisions

  • wiki/sources.html is canonical for generated Hyperwiki projects. Do not create wiki/Sources.html unless a compatibility migration explicitly asks for it.
  • wiki/AGENTS.html is the visible wiki guide. Root AGENTS.md remains the agent entrypoint for CLI and editor tools.
  • Generated pages use Hyperwiki's shared wiki stylesheet through /assets/wiki.css. Standalone embedded-CSS HTML belongs to custom artifacts or future export flows.
  • Hyperwiki projects are local-first: repo files, Git state, terminal sessions, credentials, and environment variables stay inside the developer's local trust boundary.

Skill Compatibility

When an agent uses an external project wiki skill, apply the planning, source, log, and audit workflow while preserving these Hyperwiki scaffold names and app-serving conventions.
