<h1>hyperwiki</h1>
<p>hyperwiki is Localhost Tooling for docs-driven agentic development.</p>
<p>It runs as a local app over the developer's own machine, repo files, Git state, terminal sessions, credentials, and environment variables, keeping that local environment as the trust boundary.</p>
<section>
  <h2>Current Focus</h2>
<p>The MVP now ships as a Tauri desktop app with a Rust-owned local runtime, Vite React app shell, Tailwind v4 and shadcn/ui components, MDX wiki scaffold, local dev workspace, xterm-backed terminal panels, visible Localhost Tooling guardrails, Rust CLI commands, multi-project switching, direct project import and removal, plan command actions, plan-scoped terminals, in-app worktree creation, global Settings for theme, Soul, Memory, and AGENTS.md sync, and provider-aware local agent terminals. The <a href="/wiki/plans/features/react-shadcn-mdx-rewrite.mdx">React, shadcn, and MDX Rewrite</a> is complete and merged into local <code>main</code>; pushing still requires approval.</p>
</section>
