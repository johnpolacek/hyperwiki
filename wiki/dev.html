Desktop Runtime

  • The canonical app runtime is the Tauri bundle at src-tauri/target/release/bundle/macos/Hyperwiki.app.
  • The Rust binary provides compatibility commands for init, reset, dev, launch, and mcp.
  • The legacy Node server remains in the repo as migration reference and smoke-test coverage, not as the shipping desktop backend.
  • Browser assets required by the Tauri bundle must live under public/, including vendored xterm files under public/vendor/@xterm/.

Portless Preview

  • Main preview: https://hyperwiki.localhost.
  • Feature worktree preview: https://<branch-slug>.hyperwiki.localhost.
  • pnpm dev remains the local development preview for this repository and currently runs the legacy Node reference server through Portless.

Parallel Worktrees

pnpm wt:doctor
pnpm wt:create feature/my-change
pnpm wt:list
pnpm wt:resume feature/my-change
pnpm wt:open feature/my-change
pnpm wt:finish feature/my-change
pnpm wt:prune

Feature worktrees live under ../hyperwiki.worktrees/<branch-slug>. The finish policy is merge, preserving feature branch history.

Packaging Notes

pnpm run tauri:build currently verifies the macOS .app target. DMG generation is intentionally deferred until signing/notarization and installer diagnostics are handled as release work.

Workflow

Plan meaningful work in plans, keep source truth in repo files and Git, and use the dev workspace to inspect wiki pages and terminal sessions.
