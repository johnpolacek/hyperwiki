2026-05-14 source decision | Localhost Tooling identity

  • Decision: hyperwiki should fully claim and identify as Localhost Tooling.
  • Product boundary: hyperwiki runs as a local app over the developer's own machine, repo files, Git state, terminal sessions, credentials, and environment variables.
  • Implication: hosted sync, telemetry, remote execution, and cloud-backed memory are not default assumptions; local repo truth and visible runtime boundaries remain the product thesis.
