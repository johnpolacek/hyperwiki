## Phase goal

Mock every main flow from the user stories using the Phase 2 components — surfacing gaps that feed back into the design system, and extracting recurring sections into reusable **blocks**.

## Entry gate

Phase 2 cleared (design system in place).

## Sub-agent contract

- **Skills**: `frontend-design` and `make-interfaces-feel-better`.
- **Produces**: mock screens (specified with `Screen`/`Mockup` in the sub-plan), updates to the design-system components where mocks expose gaps, and a set of composable blocks. Tracked in `/wiki/plans/ui-mocks/`.
