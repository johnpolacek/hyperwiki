<section>
  <h2>Intent</h2>
  <p>Improve hyperwiki's agent-readable local wiki surface using the Holocron assessment's conservative recommendations while preserving Tauri/Rust ownership, repo-local files, and local-only trust boundaries.</p>
</section>

<Scope>
  <ul>
    <li>Expose per-page Markdown through a local API alias that is explicit about Markdown output.</li>
    <li>Add a local <code>llms.txt</code>-style wiki export assembled from repo-visible MDX pages and their Markdown derivatives.</li>
    <li>Enrich the Rust wiki page index with parsed frontmatter, headings, wiki links, and validation warnings for unresolved internal links.</li>
    <li>Add an app-visible copy-as-Markdown action for React-rendered plan pages.</li>
  </ul>
</Scope>

<section>
  <h2>Non-Goals</h2>
  <ul>
    <li>No Holocron runtime dependency, Spiceflow/RSC shell, deploy target, hosted docs model, or <code>docs.json</code> canonical structure.</li>
    <li>No arbitrary MDX/TSX imports or full AST pipeline in this unit.</li>
    <li>No zip export until the page/index export model is proven useful.</li>
  </ul>
</section>
