Agent Execute Launch Guard

Summary

  • Status: complete
  • Shape: compact safety correction
  • Current unit: complete
  • Validation: pnpm run check, browser smoke

Intent

Execute should only send generated work prompts into a real configured agent process. If a project has no agent launch command, Hyperwiki should stop with visible status instead of pasting the prompt into a plain shell.

Unit 01 - Require Configured Agent Command

  • Status: complete
  • Detect missing agent launch commands before creating or reusing an agent terminal for handoff prompts.
  • Show an actionable error that tells the user to configure agent.launchCommand.
  • Add smoke coverage for the missing-command guard.
