<section>
  <h2>Intent</h2>
  <p>When Execute Unit is clicked while a visible agent terminal is already open, hyperwiki should ask whether to reuse that terminal, start a new agent, or cancel. The selected visible terminal is preferred; if none is selected, the first visible agent terminal is offered.</p>
</section>
<section>
  <h2>Implementation</h2>
  <ul>
    <li>Detect visible live command-backed agent sessions before Execute submits the unit prompt.</li>
    <li>Show a confirmation with <code>Run in current agent</code>, <code>New agent</code>, and <code>Cancel</code>.</li>
    <li>When reusing, retarget the session metadata to the resolved Execute unit scope and submit the prompt to that exact <code>sessionId</code>.</li>
    <li>When starting new, preserve the existing hidden general standby promotion and refill behavior.</li>
  </ul>
</section>
