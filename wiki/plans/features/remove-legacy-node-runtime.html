Summary

Fixed Decisions

  • package.json remains only for pnpm developer scripts, the Tauri CLI, frontend static checks, and asset smoke tests.
  • The product command surface is the Rust/Tauri binary. No npm bin shim remains.
  • Worktree commands stay convenient through pnpm wt:*, but those scripts invoke Rust CLI commands instead of a Node shim.
  • The frontend uses Tauri command transport only. HTTP and WebSocket transports are no longer supported for Hyperwiki's own app shell.

  4. Add Rust CLI worktree subcommands for the repo-local worktree workflow.
  5. Replace current docs and wiki source truth that still describe Node as a live reference runtime.
