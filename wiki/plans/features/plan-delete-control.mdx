<PlanUnit title="Unit 01 - Plan delete affordance" status="complete">
  <p>Top-level plan pages should expose a compact destructive action next to Copy. Clicking the X should show inline Cancel and Delete Plan actions. Stage, unit, and structural index pages must not show the control.</p>
  <p>The backend should delete only validated plan roots and their associated child plan directory when present, refusing indexes, stages, units, and unsafe paths.</p>
  <p>Complete on 2026-06-03: the React renderer shows the X and inline confirmation only when the app marks the route as a deletable plan root, and the Tauri endpoint deletes validated plan roots plus associated child trees.</p>
