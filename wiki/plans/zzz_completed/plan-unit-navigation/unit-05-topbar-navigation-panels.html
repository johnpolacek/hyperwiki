Unit 05 - TopBar Navigation Panels

Plan Unit Navigation

Summary

  • Status: complete
  • Add a workspace TopBar with a left-aligned HyperWiki icon animation and text label.
  • Add right-aligned Projects and Up Next buttons.
  • Use the Lucide arrow-big-right-dash icon for the Up Next button.
  • Open both button surfaces as a top-right panel that slides in from the top right, then fades in its UI content.
  • Verification: pnpm run check, pnpm run smoke:browser, pnpm run smoke:launch, and agent-browser visual inspection of the TopBar and panel animations.

Scope

  • Replace or consolidate existing workspace chrome so Projects and Up Next are primary TopBar actions.
  • Keep Projects and Up Next content functionally equivalent to the current controls while moving them into the new panel pattern.
  • Preserve keyboard and pointer access, visible focus states, panel close behavior, and responsive behavior on narrow viewports.

Non-goals

  • Do not redesign the plan tree or change wiki document hierarchy.
  • Do not add new project management capabilities beyond relocating the current Projects surface.
  • Do not introduce a new icon library; use Lucide for the requested icon.
