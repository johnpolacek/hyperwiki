<h2>Scope</h2>
<ul>
  <li>Replace or consolidate existing workspace chrome so Projects and Up Next are primary TopBar actions.</li>
  <li>Keep Projects and Up Next content functionally equivalent to the current controls while moving them into the new panel pattern.</li>
  <li>Preserve keyboard and pointer access, visible focus states, panel close behavior, and responsive behavior on narrow viewports.</li>
</ul>
<h2>Non-goals</h2>
<ul>
  <li>Do not redesign the plan tree or change wiki document hierarchy.</li>
  <li>Do not add new project management capabilities beyond relocating the current Projects surface.</li>
  <li>Do not introduce a new icon library; use Lucide for the requested icon.</li>
</ul>
