    <section>

      <ul>
        <li>Delete the Ideas workspace route, topbar button, dropdown, page, sidebar group, and idea files.</li>
        <li>Remove idea create/list/promote endpoints and tests.</li>
        <li>Keep project import and project creation as the sole new-workflow path.</li>
        <li>Default app-shell navigation to <code>/projects</code>.</li>
        <li>Update wiki durable context to describe project-only behavior.</li>
      </ul>
    </section>
