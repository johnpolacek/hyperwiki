Remove Ideas

Summary

  • Status: complete
  • Shape: product simplification
  • Completed: 2026-05-15 in commit 2a479b29c89af10b58d285a3be4261699273e8d2
  • Result: Ideas were removed and Projects became the only new-workspace surface.

Intent

hyperwiki should only manage projects. Imported briefs should create a project directly instead of becoming intermediate idea pages that later need promotion.

Scope

  • Delete the Ideas workspace route, topbar button, dropdown, page, sidebar group, and idea files.
  • Remove idea create/list/promote endpoints and tests.
  • Keep project import and project creation as the sole new-workflow path.
  • Default app-shell navigation to /projects.
  • Update wiki durable context to describe project-only behavior.

Completion Notes

  • Removed the Ideas workspace route, topbar item, wiki scaffold, API endpoints, smoke script, and idea files.
  • Kept imported briefs on the direct project creation path.
  • Preserved legacy /ideas and /dashboard app-shell routes by resolving them to Projects.
