Unit 02 - Sidebar Project Pin

Agent Workspace Enablers

Summary

  • Status: complete
  • Project navigation is pinned to the bottom of the sidebar and uses a dedicated styling hook.
  • Plan rows stay directly under the Plans heading, with completed plans grouped under the Completed Plans archive.
  • Verification: covered by pnpm run check and browser smoke assertions for Project placement, stacking, chevron alignment, and plan-row spacing.

Goal

Keep the Project navigation group pinned to the bottom of the workspace sidebar while preserving quick access to plan navigation above it.

Scope

  • Pin the Project group to the sidebar bottom in the workspace navigation.
  • Keep plan rows directly underneath the Plans heading whether Project is collapsed or expanded.
  • Align the Plans heading with plan chevrons, give Project a matching up/down chevron, and reduce the top gap above the Plans section.
  • When expanded, let the Project group sit above the plan navigation surface so its links remain usable near the bottom edge.
  • Update browser smoke coverage for the sidebar placement.

Verification

  • pnpm run check.
  • pnpm run smoke:browser.
