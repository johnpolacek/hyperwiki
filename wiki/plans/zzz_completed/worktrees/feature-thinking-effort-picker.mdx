        <li>Persisted the selected default in browser-local workspace state.</li>
        <li>Applied the selected level to newly opened Codex agent terminals with <code>-c model_reasoning_effort="..."</code>.</li>
        <li>Left non-Codex agent commands and existing terminal sessions unchanged.</li>
