## Stage Goal

A new lifecycle dashboard, reachable from the TopBar, renders the 6 phases with truthful status and gate state, an "Ask the Orchestrator" action, and a per-phase handoff that respects the enablement rule.

<Dependencies>

- Stage 04 (the `"orchestrate"` / `"lifecycle-phase"` dispatch the buttons call).

</Dependencies>

## Screen content & layout

Shared frame: the standard workspace header with the project switcher; the dashboard body replaces the wiki/terminal pane when `route.kind === "lifecycle"`. A top action row holds "Ask the Orchestrator". Below it, a vertical phase board.
