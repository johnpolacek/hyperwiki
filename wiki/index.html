HyperWiki

HyperWiki is a repo-local HTML workspace for docs-driven agentic development.

Current Focus

The MVP now has the single-package CLI, HTML wiki scaffold, local dev workspace, wterm-backed terminal panels, and visible local guardrails needed to dogfood HyperWiki in this repository. The next focus is a one-command launch path that starts the local server, opens the browser workspace, and creates the default wterm operating session.
