hyperwiki

hyperwiki is Localhost Tooling for docs-driven agentic development.

It runs as a local app over the developer's own machine, repo files, Git state, terminal sessions, credentials, and environment variables, keeping that local environment as the trust boundary.

Current Focus

The MVP now has the single-package CLI, HTML wiki scaffold, local dev workspace, wterm-backed terminal panels, visible Localhost Tooling guardrails, one-command launch, multi-project switching, dashboard flows, plan command actions, explicit Execute targets for main or worktree execution, and global Settings for theme, Soul, Memory, and AGENTS.md sync.
