hyperwiki

hyperwiki is a repo-local HTML workspace for docs-driven agentic development.

Current Focus

The MVP now has the single-package CLI, HTML wiki scaffold, local dev workspace, wterm-backed terminal panels, visible local guardrails, one-command launch, and multi-project switching for dogfooding hyperwiki across repositories. The current focus is Stage 07: agent-native verification loops, machine-readable project contracts, agent review workflows, and structured MCP surfaces.
