hyperwiki

hyperwiki is Localhost Tooling for docs-driven agentic development.

It runs as a local app over the developer's own machine, repo files, Git state, terminal sessions, credentials, and environment variables, keeping that local environment as the trust boundary.

Current Focus

The MVP now ships as a Tauri desktop app with a Rust-owned local runtime, HTML wiki scaffold, local dev workspace, xterm-backed terminal panels, visible Localhost Tooling guardrails, Rust CLI commands, multi-project switching, direct project import and removal, plan command actions, plan-scoped terminals, in-app worktree creation, global Settings for theme, Soul, Memory, and AGENTS.md sync, and provider-aware local agent terminals. The active current plan is Project Plan Creation.
